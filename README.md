## 🌍 Overview

**ANNADAAN** is a purpose-driven platform designed to reduce food waste by creating a seamless connection between food donors, NGOs, and volunteers.

Instead of letting surplus food go to waste, ANNADAAN ensures it is redistributed efficiently to communities in need — powered by real-time technology and intuitive design.

---

## ✨ Key Features

- 🔐 Secure Authentication using Firebase  
- 📦 Food Listing System for donors  
- 👥 Role-Based Dashboards (Donor / NGO / Volunteer)  
- 🗺️ Live Map Tracking using Leaflet  
- ⚡ Real-Time Updates with Firestore  
- 🤖 AI Assistance for donors using Gemini API  
- 🎨 Modern UI/UX powered by Tailwind CSS  

---

## 🧠 How It Works

1. Donors list surplus food  
2. NGOs receive and manage requests  
3. Volunteers pick up and deliver food  
4. Recipients benefit — zero waste, maximum impact
